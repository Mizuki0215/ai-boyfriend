# AI 男朋友 Telegram Bot 🤖💕

一個有記憶、有性格嘅 AI 男朋友，可以喺 Telegram 同你傾偈！

## 功能
- ✅ 記得你所有對話
- ✅ 有性格設定（溫柔體貼）
- ✅ 支援 /start, /help, /reset 指令
- ✅ 每個用戶獨立記憶

## 安裝步驟
1. 複製 repository
2. 建立 virtual environment: `python3 -m venv ai_venv`
3. 激活: `source ai_venv/bin/activate`
4. 安裝套件: `pip install -r requirements.txt`
5. 去 @BotFather 拎 token，改喺 telegram_bot.py
6. 行: `python telegram_bot.py`

## 使用
- /start - 開始對話
- /help - 睇說明
- /reset - 重置記憶
